help Como Usar o Conversor

Bem-vindo! Siga este guia rápido para utilizar todas as ferramentas avançadas.

1. Converter Formatos

Selecione "Converter Formato", arraste seus arquivos, escolha o formato de saída (suportando PDF, DOCX, TXT, planilhas CSV, XLSX, XLS, dados Parquet, apresentações PPT/PPTX e exportação de GIFs animados a partir de vídeos) e clique em Iniciar. Ideal para conversões universais individuais.

2. Juntar PDFs

Selecione "Juntar PDFs" e arraste múltiplos PDFs. O sistema unirá todos em um único documento, mantendo a ordem da fila de arquivos.

3. Dividir PDF

Arraste um PDF e selecione "Dividir PDF". Cada página virará um arquivo próprio, e você receberá um arquivo `.zip` contendo as páginas fatiadas.

4. Imagens para PDF e PDF para Imagens

Arraste várias imagens (PNG, JPG) e selecione "Imagens para PDF" para construir um portfólio. Ou arraste um PDF e use "PDF para Imagens" para receber páginas em alta resolução comprimidas num ZIP.

5. Segurança de PDFs

Com as opções "Proteger PDF" e "Desbloquear PDF", o campo de senha aparecerá. Você pode blindar um documento usando criptografia AES-256 localmente, ou remover travas existentes se possuir a senha original.

6. Extrair Tabelas (PDF)

Selecione "Extrair Tabelas (PDF)" para ler dados tabulares de dentro do PDF e convertê-los automaticamente para uma planilha .xlsx ou .csv consolidada.

7. Sanitizar Arquivo

Use "Sanitizar Arquivo" em PDFs ou Imagens para limpar todos os metadados (incluindo EXIF, autor, software usado), garantindo total privacidade antes do compartilhamento.

8. Comprimir Arquivo

Use "Comprimir Arquivo" para reduzir o peso de PDFs (removendo lixo interno) ou para recomprimir vídeos pesados usando aceleração FFmpeg, gerando arquivos mais leves sem perder qualidade.

9. Reconhecimento OCR

Selecione "Reconhecimento OCR" para escanear visualmente PDFs fatiados ou imagens, extraindo o texto e transformando-o num documento editável estruturado (como .docx ou .txt).

10. Edição Avançada de PDFs

No painel lateral, você encontra ferramentas específicas para manipulação rápida de documentos PDF:

  • Rotacionar PDF: Corrige a orientação de todas as páginas do documento (suporta 90°, 180° e 270°).
  • Remover Páginas: Exclui páginas específicas do PDF. Digite os números das páginas ou intervalos separados por vírgula (ex: 1, 3-5, 8).
  • Numerar Páginas: Insere automaticamente o número da página (no formato "Página X de Y") no rodapé de cada folha.
  • Marca d'Água: Adiciona um texto semitransparente personalizado centralizado em diagonal sobre todas as páginas do PDF.
  • Extrair Páginas: Cria um novo PDF contendo apenas as páginas indicadas (ex: 1, 3, 5-8).
  • Reparar PDF: Reconstrói e recupera a árvore de páginas e a tabela de referências cruzadas de arquivos PDF danificados ou corrompidos.

11. Censura e Assinatura Digital

  • Censurar PDF (Tarja Preta): Localiza nomes, números, CPFs ou palavras-chave no texto do PDF e insere uma tarja preta de censura definitiva (ideal para conformidade com a LGPD).
  • Assinar PDF (A1): Aplica uma assinatura digital no PDF usando o seu Certificado Digital A1 corporativo (arquivos .pfx ou .p12).

12. Converter em Lote (ZIP)

Selecione "Converter Lote (ZIP)" para enviar múltiplos arquivos de uma vez. O motor converterá todos de forma paralela e os agrupará em um arquivo compactado .zip para download rápido.

13. Visualização (Preview) de Arquivos

Ao arrastar ou selecionar arquivos para a aplicação, a área central se transforma em um Visualizador Dinâmico Inteligente. Formatos comuns como imagens, vídeos, áudios e PDFs são exibidos instantaneamente utilizando a memória RAM local. Formatos corporativos do Office (Word .docx, planilhas Excel .xlsx/.xls e arquivos delimitados .csv) contam com renderização dinâmica gerada sob demanda localmente pelo backend offline.

14. Fatiar por MB (SEI)

Esta ferramenta divide um arquivo PDF individual em partes menores baseadas no tamanho limite em Megabytes definido pelo usuário. Ideal para envio em sistemas com limitação estrita de tamanho por arquivo, como o SEI ou portais de peticionamento judicial.

15. Renomear em Lote

Permite renomear múltiplos arquivos simultaneamente de forma automatizada com base em um padrão configurável. Suporta o uso de tags dinâmicas como {nome} (nome original sem extensão), {nome_original} (nome completo com extensão) e {i} (índice sequencial iniciando em 1), retornando um pacote compactado .zip com os arquivos renomeados.
