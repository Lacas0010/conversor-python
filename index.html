help Como Usar o Conversor

Bem-vindo! Siga este guia rápido para utilizar todas as ferramentas avançadas.

1. Converter Formatos

Selecione "Converter Formato", arraste seus arquivos, escolha o formato de saída (suportando PDF, DOCX, TXT, planilhas CSV, XLSX, XLS, dados Parquet, apresentações PPT/PPTX e exportação de GIFs animados a partir de vídeos) e clique em Iniciar. Ideal para conversões universais individuais.

2. Juntar PDFs

Selecione "Juntar PDFs" e arraste múltiplos PDFs. O sistema unirá todos em um único documento, mantendo a ordem da fila de arquivos.

3. Dividir PDF

Arraste um PDF e selecione "Dividir PDF". Cada página virará um arquivo próprio, e você receberá um arquivo `.zip` contendo as páginas fatiadas.

4. Imagens para PDF e PDF para Imagens

Arraste várias imagens (PNG, JPG) e selecione "Imagens para PDF" para construir um portfólio. Ou arraste um PDF e use "PDF para Imagens" para receber páginas em alta resolução comprimidas num ZIP.

5. Segurança de PDFs

Com as opções "Proteger PDF" e "Desbloquear PDF", o campo de senha aparecerá. Você pode blindar um documento usando criptografia AES-256 localmente, ou remover travas existentes se possuir a senha original.

6. Extrair Tabelas (PDF)

Selecione "Extrair Tabelas (PDF)" para ler dados tabulares de dentro do PDF e convertê-los automaticamente para uma planilha .xlsx ou .csv consolidada.

7. Sanitizar Arquivo

Use "Sanitizar Arquivo" em PDFs ou Imagens para limpar todos os metadados (incluindo EXIF, autor, software usado), garantindo total privacidade antes do compartilhamento.

8. Comprimir Arquivo

Use "Comprimir Arquivo" para reduzir o peso de PDFs (removendo lixo interno) ou para recomprimir vídeos pesados usando aceleração FFmpeg, gerando arquivos mais leves sem perder qualidade.

9. Reconhecimento OCR

Selecione "Reconhecimento OCR" para escanear visualmente PDFs fatiados ou imagens, extraindo o texto e transformando-o num documento editável estruturado (como .docx ou .txt).

10. Converter em Lote (ZIP)

Selecione "Converter Lote (ZIP)" para enviar múltiplos arquivos de uma vez. O motor converterá todos de forma paralela e os agrupará em um arquivo compactado .zip para download rápido.
